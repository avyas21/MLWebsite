K-Nearest Neighbors

K-Nearest Neighbors (KNN) is a supervised classification algorithm which predicts the class for an input based on the data the input is closest to.
It looks at the k closest points, and returns the class that is most common among those.

In the below simulation, we have a k-nearest neighbors model for the xy plane. You can enter a list of data points and the number of closest points you want to check to predict an input's class. You can then enter points to test and see the returned class based on the KNN algorithm.

Ex: Insert Points: (0,1,A);(1,2,A);(1,3,A);(1,4,A);(2,5,A);(10,11,B);(10,14,B);(14,10,B);(15,13,B);(10,12,B), k: 3
